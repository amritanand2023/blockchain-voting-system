# Blockchain-Based Organization Voting System

This project implements a secure, transparent, and decentralized voting system using blockchain technology. It is designed for organization-level elections where integrity and trust are important.

---

##  Features

- Decentralized voting using blockchain
- Smart contract-based voting logic
- One person, one vote enforcement
- Transparent and tamper-proof data storage
- Public audit system with blockchain logs
- Real-time result display

---

##  Technologies Used

- Ethereum Blockchain
- Solidity (Smart Contracts)
- Web3.js
- MetaMask
- Ganache
- HTML, CSS, JavaScript
---

##  How It Works

1. Admin connects wallet and adds candidates  
2. Admin starts the election  
3. Users connect MetaMask and cast votes  
4. Votes are stored on blockchain  
5. Duplicate voting is prevented  
6. Public audit page shows logs  
7. Results are displayed after election ends

##  How to Run

1. Start Ganache (local blockchain)  
2. Connect MetaMask to Ganache network  
3. Deploy smart contract using Remix IDE  
4. Update contract address in frontend files  
5. Open `home.html` in browser
